# dev-env

Terminal config: zsh, starship, ghostty, git.

## Install

```bash
git clone git@github.com:IFAKA/dev-env.git ~/dev-env
cd ~/dev-env && ./setup.sh install
```

With Neovim config (VimZap):

```bash
./setup.sh install --with-vimzap
```

## Uninstall (clean, no traces)

```bash
./setup.sh uninstall
```

## Status

```bash
./setup.sh status
```
